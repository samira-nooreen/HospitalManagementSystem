# Hospital Management System

A Java-based Hospital Management System using JDBC and MySQL.

## Features
- Add and view patients
- View doctors
- Book appointments
- Doctor availability check

## Technologies Used
- Java
- JDBC
- MySQL
- IntelliJ IDEA

## How to Run
1. Create a MySQL database named `hospital`
2. Create required tables (`patients`, `doctors`, `appointments`)
3. Update database credentials in code
4. Run `HospitalManagementSystem.java`
